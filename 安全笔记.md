php变量规则
```
1. 变量以$符号开头，后面是变量名称
2. 变量名称必须以字母或下划线开头，不能以数字开头
3. 变量名称可以包含字母、数字和下划线
4. 变量名称大小写敏感($a和$A是两个变量)
```

变量引用
```php
<?php
$name = 'Tom';
$a = &$name;
echo $a; //Tom
?>
```

输出信息 echo print_r var_dump
数据类型：字符串(单引号 双引号)、整型、浮点型、逻辑(与或非异或)、数组、对象、

字符串函数：http://www.w3school.com.cn/php/php_ref_string.asp
strlen()函数返回字符串的长度，以字符计。
strpos()函数用于检索定位字符串内指定的字符或文本。

PHP常量：常量名没有$，第三个参数规定常量名是否大小写不敏感，默认False(敏感)，例：define("NAME",'Mike',True); echo name;

PHP运算符： $x+$y   $x-$y   $x*$y   $x/$y   $x%$y   $x++    $x--
PHP 逻辑运算符：and  or  xor  &&  ||  !
注意PHP运算符的优先级，注意：&&优先级高于and

PHP判断与循环
